# SimpleTags

Simple declarative HTML DSL, inspired by (Deklarativna)[https://github.com/dlitvakb/deklarativna].

## Installation

Via npm on Node:

```
npm install simpletags
```

Reference in your program:

```js
var simpletags = require('simpletags');
```
